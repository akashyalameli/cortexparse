# CortexParse

CortexParse is a privacy-first, AI-native intelligent document understanding platform powered by multimodal LLM agents and LangGraph workflows.

## Features

- AI-native document understanding
- Multimodal LLM extraction
- Adaptive retry workflows
- Confidence-based validation
- Human-in-the-loop review
- Privacy-first architecture
- LangGraph orchestration
- RabbitMQ integration
- MinIO object storage
- Ollama local LLM hosting

## Tech Stack

- FastAPI
- LangGraph
- Ollama
- Qwen2.5-VL
- React + Vite
- RabbitMQ
- MinIO
- Docker Compose
